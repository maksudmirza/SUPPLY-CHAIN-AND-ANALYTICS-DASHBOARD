
# Supply Chain & Inventory Analytics Dashboard | Power BI

This project is an interactive **Power BI dashboard** built using the **DataCo Smart Supply Chain Dataset** to analyze sales, customers, products, shipping performance, and geographic insights. The dashboard helps businesses monitor key performance indicators (KPIs), identify trends, optimize logistics, and make data-driven decisions.

# Features

* Executive Summary Dashboard
* Customer Analytics
* Product Analytics
* Shipping Analytics
* Geographic Analysis
* Interactive Slicers & Drill-down
* Star Schema Data Model
* DAX Measures & KPIs
* Responsive and Professional Dashboard Design

# Key KPIs

* Total Sales
* Total Profit
* Total Orders
* Profit Margin
* Customer Count
* Average Sales per Customer
* Late Deliveries
* On-Time Delivery %
* Average Delivery Days
* Countries Served
* Top Market

# Tools & Technologies

* Microsoft Power BI
* Power Query
* DAX (Data Analysis Expressions)
* Data Modeling (Star Schema)
* DataCo Smart Supply Chain Dataset

# Dashboard Pages

1. Executive Summary
2. Customer Analytics
3. Product Analytics
4. Shipping Analytics
5. Geographic Analysis

# Business Insights

* Monitors overall business performance through KPIs.
* Identifies top-performing customers and products.
* Evaluates shipping efficiency and delivery performance.
* Analyzes sales across markets, regions, and countries.
* Supports strategic decision-making with interactive visualizations.
